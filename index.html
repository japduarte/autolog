Guardar & Restaurar
💾 Exportar dados
Descarrega um ficheiro autolog.json com todos os teus dados. Guarda-o na pasta que quiseres no telemóvel.
📂 Importar dados
Carrega um ficheiro autolog.json previamente exportado. Os dados actuais serão substituídos.
🌐 Instalar como App
No Chrome Android: toca nos ⋮ três pontos → "Adicionar ao ecrã inicial".
A app fica instalada como ícone. Os dados ficam guardados automaticamente neste browser.
🗑 Apagar tudo
Remove todos os carros e registos permanentemente.
